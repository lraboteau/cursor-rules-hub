# SEO Fundamentals

- Ensure each indexable page has a unique, descriptive `<title>` and meta description.
- Use semantic HTML headings (`h1` to `h6`) with a clear information hierarchy.
- Publish canonical URLs and prevent duplicate-content variants from being indexed.
- Implement structured data (JSON-LD) for relevant entities and validate it regularly.
- Optimize performance signals that affect search visibility, especially Core Web Vitals.
- Keep robots directives and sitemap coverage consistent with crawl/indexing intent.
