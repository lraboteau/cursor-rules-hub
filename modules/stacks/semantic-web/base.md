# Semantic Web Baseline

- Model data with explicit vocabularies and stable IRIs to preserve long-term interoperability.
- Keep JSON-LD, RDF, and schema definitions aligned across documentation, examples, and production payloads.
- Prefer standards-compliant serialization and parsing behavior over implementation-specific shortcuts.
- Document context usage, term definitions, and compatibility guarantees for downstream consumers.
- Run semantic validation checks before release.
